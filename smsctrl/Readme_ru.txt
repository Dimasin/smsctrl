SMSCTRL -

Программа предназначена для мониторинга удаленного хоста методом проверки возможности TCP соединения на выбранный порт. В случае невозможности установить соединение заданное число раз программа может отправить СМС сообщение. Возможна отправка повторного СМС сообщения через устанавливаемый интервал времени. Это можно использовать для управления GSM электрической розеткой по СМС. Функция отправки СМС работает только с оператором МТС.
Для использования функции отправки СМС необходимо установить программу от МТС и сделать указанные настройки телефона. Программа и инструкции находятся на сайте МТС: http://www.mts.ru/mob_connect/messaging/sms/sending_sms/pcm/ 

Предложения и замечания приветствуются.
